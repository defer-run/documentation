## Endpoint

GET https://api.defer.run/public/v1/executions/{execution-id}

## Request

### Path Parameters

<ParamField path="execution-id" type="string" required>
  The execution ID.
</ParamField>

## Response

### Body

<ResponseField name="id" type="string" required>
  The state of the execution.
</ResponseField>

<ResponseField
  name="state"
  type="created | running | succeed | failed | aborting | aborted | cancelled | cancelling | started"
  required
>
  The ID of the execution.
</ResponseField>

<ResponseField name="result" type="any">
  The result returns by the function.
</ResponseField>
